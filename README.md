# IoTMqttPLC - AC PLC Simulator

This is an Air Conditioner Programmable Logic Controller Simulator IoT project.

For this, you will use:

- Visual Studio Code

  - https://code.visualstudio.com

- Wokwi Simulator extension for VS Code

  - https://wokwi.com
  - https://docs.wokwi.com/vscode/getting-started
  - https://marketplace.visualstudio.com/items?itemName=wokwi.wokwi-vscode

- PlatformIO IDE extension for VS Code

  - https://platformio.org
  - https://docs.platformio.org/en/latest/integration/ide/vscode.html
  - https://marketplace.visualstudio.com/items?itemName=platformio.platformio-ide

- ESP32 microcontroller development board

  - https://wokwi.com/esp32
  - https://docs.wokwi.com/guides/esp32
  - https://docs.platformio.org/en/latest/boards/espressif32/esp32dev.html

- PubSubClient, client library for MQTT messaging protocol

  - https://docs.wokwi.com/guides/libraries
  - https://www.arduinolibraries.info/libraries/pub-sub-client

- C++

  - https://code.visualstudio.com/docs/languages/cpp
  - https://code.visualstudio.com/docs/cpp/introvideos-cpp
  - https://marketplace.visualstudio.com/items?itemName=ms-vscode.cpptools
